HTML Cheat Sheet

Content Sectioning

Tag Name Description
<header> Header Represents introductory content, typically a group of introductory or navigational aids.
<footer> Footer Represents a footer for its nearest sectioning content or sectioning root element.
<main> Main Represents the dominant content of the body of a document.
<nav> Navigation Represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents.
<section> Section Represents a generic standalone section of a document, which doesn't have a more specific semantic element to represent it.
<h1> <h2> <h3> <h4> <h5> <h6> Heading Elements Represent six levels of section headings. <h1> is the highest section level and <h6> is the lowest.

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Text Content

Tag Name Description
<p> Paragraph Represents a paragraph.
<div> Content Division The generic container for flow content. It has no effect on the content or layout until styled in some way using CSS.
<ul> Unordered List Represents an unordered list of items, typically rendered as a bulleted list.
<ol> Ordered List Represents an ordered list of items â€” typically rendered as a numbered list.
<li> List Item Represents an item in a list. It must be contained in a parent element: an ordered list (ol) or an unordered list (ul).

Inline Text Semantics

Tag Name Description
<a> Anchor Creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.
<br> Content Division Produces a line break in text.
<em> Emphasis Marks text that has stress emphasis.
<strong> Strong Importance Indicates that its contents have strong importance, seriousness, or urgency. Browsers typically render the contents in bold type.
<span> Content Span Generic inline container for phrasing content, which does not inherently represent anything. It can be used to group elements for styling purposes.

Image and Multimedia

Tag Name Description
<audio> Embed Audio Used to embed sound content in documents.
<img> Image Embed Embeds an image into the document.
<video> Video Embed Embeds a media player which supports video playback into the document.
